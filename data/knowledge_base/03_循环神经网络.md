# 循环神经网络（RNN）

## 1. RNN 基本原理

循环神经网络（Recurrent Neural Network, RNN）是一类专门用于处理序列数据的神经网络。与传统的前馈神经网络不同，RNN 具有"记忆"能力，能够通过隐藏状态（Hidden State）捕获序列中的时序信息。

### 网络结构

RNN 在每个时间步共享相同的参数，通过循环连接将前一时刻的隐藏状态传递到当前时刻。这种结构使得 RNN 能够处理任意长度的序列。

### 核心公式

对于时间步 \(t\)，RNN 的计算如下：

\[
\mathbf{h}_t = \tanh(\mathbf{W}_{hh} \mathbf{h}_{t-1} + \mathbf{W}_{xh} \mathbf{x}_t + \mathbf{b}_h)
\]

\[
\mathbf{y}_t = \mathbf{W}_{hy} \mathbf{h}_t + \mathbf{b}_y
\]

其中：
- \(\mathbf{x}_t\)：时间步 \(t\) 的输入
- \(\mathbf{h}_t\)：时间步 \(t\) 的隐藏状态
- \(\mathbf{y}_t\)：时间步 \(t\) 的输出
- \(\mathbf{W}_{xh}, \mathbf{W}_{hh}, \mathbf{W}_{hy}\)：权重矩阵
- \(\mathbf{h}_0\)：通常初始化为零向量

```python
import numpy as np

class SimpleRNN:
    """简单 RNN 实现（NumPy）"""
    def __init__(self, input_dim, hidden_dim, output_dim):
        # 初始化权重
        self.W_xh = np.random.randn(hidden_dim, input_dim) * 0.01
        self.W_hh = np.random.randn(hidden_dim, hidden_dim) * 0.01
        self.W_hy = np.random.randn(output_dim, hidden_dim) * 0.01
        self.b_h = np.zeros((hidden_dim, 1))
        self.b_y = np.zeros((output_dim, 1))
        self.hidden_dim = hidden_dim
    
    def forward(self, x_sequence):
        """
        前向传播
        x_sequence: 形状为 (seq_len, input_dim) 的序列
        """
        seq_len = x_sequence.shape[0]
        h = np.zeros((self.hidden_dim, 1))  # 初始隐藏状态
        outputs = []
        hidden_states = []
        
        for t in range(seq_len):
            x_t = x_sequence[t].reshape(-1, 1)
            # 更新隐藏状态
            h = np.tanh(np.dot(self.W_xh, x_t) + 
                        np.dot(self.W_hh, h) + self.b_h)
            # 计算输出
            y_t = np.dot(self.W_hy, h) + self.b_y
            outputs.append(y_t)
            hidden_states.append(h.copy())
        
        return outputs, hidden_states

# 示例使用
rnn = SimpleRNN(input_dim=10, hidden_dim=32, output_dim=5)
x_seq = np.random.randn(20, 10)  # 序列长度20，输入维度10
outputs, hidden_states = rnn.forward(x_seq)
print(f"输出序列长度: {len(outputs)}")
print(f"每个输出形状: {outputs[0].shape}")  # (5, 1)
```

### 局限性：长期依赖问题

由于梯度在反向传播过程中需要经过多个时间步，RNN 容易出现梯度消失或梯度爆炸问题，导致难以捕获长距离的依赖关系。这也是 LSTM 和 GRU 被提出的原因。

## 2. LSTM（长短期记忆网络）

LSTM（Long Short-Term Memory）由 Hochreiter 和 Schmidhuber 于 1997 年提出，通过引入门控机制有效地解决了 RNN 的长期依赖问题。

### 核心思想

LSTM 通过三个门（输入门、遗忘门、输出门）来控制信息的流动，从而实现对长期信息的记忆和遗忘。

### 公式详解

\[
\mathbf{f}_t = \sigma(\mathbf{W}_f [\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_f) \quad \text{(遗忘门)}
\]

\[
\mathbf{i}_t = \sigma(\mathbf{W}_i [\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_i) \quad \text{(输入门)}
\]

\[
\tilde{\mathbf{C}}_t = \tanh(\mathbf{W}_C [\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_C) \quad \text{(候选记忆)}
\]

\[
\mathbf{C}_t = \mathbf{f}_t \odot \mathbf{C}_{t-1} + \mathbf{i}_t \odot \tilde{\mathbf{C}}_t \quad \text{(记忆更新)}
\]

\[
\mathbf{o}_t = \sigma(\mathbf{W}_o [\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_o) \quad \text{(输出门)}
\]

\[
\mathbf{h}_t = \mathbf{o}_t \odot \tanh(\mathbf{C}_t) \quad \text{(隐藏状态)}
\]

其中 \(\odot\) 表示逐元素乘法（Hadamard 积），\(\sigma\) 为 Sigmoid 激活函数。

```python
import torch
import torch.nn as nn

# 使用 PyTorch 内置的 LSTM
lstm = nn.LSTM(
    input_size=128,      # 输入特征维度
    hidden_size=256,     # 隐藏状态维度
    num_layers=2,        # LSTM 层数
    batch_first=True,    # 输入形状为 (batch, seq, feature)
    dropout=0.2,        # 层间 dropout
    bidirectional=True    # 双向 LSTM
)

# 前向传播示例
batch_size = 16
seq_len = 50
input_dim = 128
X = torch.randn(batch_size, seq_len, input_dim)
output, (h_n, c_n) = lstm(X)
print(f"输出形状: {output.shape}")       # (16, 50, 512) 双向所以512=256*2
print(f"最后时刻隐藏状态: {h_n.shape}")  # (4, 16, 256) 2层*2方向
```

```python
# 手动实现 LSTM 单元（简化版）
class LSTMCell:
    def __init__(self, input_dim, hidden_dim):
        self.input_dim = input_dim
        self.hidden_dim = hidden_dim
        # 合并四个门的参数
        self.W = np.random.randn(4 * hidden_dim, input_dim + hidden_dim) * 0.01
        self.b = np.zeros((4 * hidden_dim, 1))
    
    def forward(self, x, h_prev, c_prev):
        """
        x: (input_dim, 1)
        h_prev: (hidden_dim, 1)
        c_prev: (hidden_dim, 1)
        """
        combined = np.concatenate([h_prev, x], axis=0)
        gates = np.dot(self.W, combined) + self.b
        
        # 分割四个门
        f = self._sigmoid(gates[:self.hidden_dim])
        i = self._sigmoid(gates[self.hidden_dim:2*self.hidden_dim])
        c_tilde = np.tanh(gates[2*self.hidden_dim:3*self.hidden_dim])
        o = self._sigmoid(gates[3*self.hidden_dim:])
        
        # 更新记忆细胞
        c = f * c_prev + i * c_tilde
        h = o * np.tanh(c)
        return h, c
    
    def _sigmoid(self, x):
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))
```

## 3. GRU（门控循环单元）

GRU（Gated Recurrent Unit）由 Cho 等人于 2014 年提出，是 LSTM 的简化版本。GRU 将 LSTM 中的遗忘门和输入门合并为更新门，并将记忆细胞和隐藏状态合并，减少了参数数量，计算效率更高。

### GRU 公式

\[
\mathbf{z}_t = \sigma(\mathbf{W}_z [\mathbf{h}_{t-1}, \mathbf{x}_t]) \quad \text{(更新门)}
\]

\[
\mathbf{r}_t = \sigma(\mathbf{W}_r [\mathbf{h}_{t-1}, \mathbf{x}_t]) \quad \text{(重置门)}
\]

\[
\tilde{\mathbf{h}}_t = \tanh(\mathbf{W}_h [\mathbf{r}_t \odot \mathbf{h}_{t-1}, \mathbf{x}_t]) \quad \text{(候选隐藏状态)}
\]

\[
\mathbf{h}_t = (1 - \mathbf{z}_t) \odot \mathbf{h}_{t-1} + \mathbf{z}_t \odot \tilde{\mathbf{h}}_t \quad \text{(隐藏状态更新)}
\]

```python
import torch.nn as nn

# PyTorch 中的 GRU
gru = nn.GRU(
    input_size=128,
    hidden_size=256,
    num_layers=2,
    batch_first=True,
    dropout=0.3
)

X = torch.randn(16, 50, 128)
output, h_n = gru(X)
print(f"GRU 输出形状: {output.shape}")  # (16, 50, 256)
print(f"GRU 最后时刻隐藏状态: {h_n.shape}")  # (2, 16, 256)
```

### LSTM vs GRU 对比

| 特性 | LSTM | GRU |
|------|------|-----|
| 门的数量 | 3（输入、遗忘、输出） | 2（更新、重置） |
| 参数数量 | 更多 | 更少 |
| 计算速度 | 较慢 | 较快 |
| 记忆细胞 | 有独立记忆细胞 | 无，隐藏状态即记忆 |
| 适用场景 | 复杂序列任务 | 资源受限或序列较短 |

## 4. 序列建模应用

### 语言模型

给定前 \(t-1\) 个词，预测第 \(t\) 个词的概率分布：

\[
P(w_t | w_1, w_2, \ldots, w_{t-1})
\]

```python
class LanguageModelRNN(nn.Module):
    """基于 RNN 的语言模型"""
    def __init__(self, vocab_size, embed_dim, hidden_dim):
        super(LanguageModelRNN, self).__init__()
        self.embedding = nn.Embedding(vocab_size, embed_dim)
        self.lstm = nn.LSTM(embed_dim, hidden_dim, num_layers=2, 
                           batch_first=True, dropout=0.3)
        self.fc = nn.Linear(hidden_dim, vocab_size)
        self.dropout = nn.Dropout(0.3)
    
    def forward(self, x, hidden=None):
        # x: (batch, seq_len)
        x = self.embedding(x)           # (batch, seq_len, embed_dim)
        x, hidden = self.lstm(x, hidden)
        x = self.dropout(x)
        logits = self.fc(x)             # (batch, seq_len, vocab_size)
        return logits, hidden

# 示例使用
model = LanguageModelRNN(vocab_size=10000, embed_dim=300, hidden_dim=512)
X = torch.randint(0, 10000, (8, 30))  # (batch=8, seq_len=30)
logits, _ = model(X)
print(f"语言模型输出形状: {logits.shape}")  # (8, 30, 10000)
```

### 机器翻译（Seq2Seq）

Seq2Seq（Sequence-to-Sequence）模型由编码器（Encoder）和解码器（Decoder）组成，编码器将源语言序列编码为固定长度的向量，解码器根据该向量生成目标语言序列。

### 时间序列预测

RNN/LSTM/GRU 广泛应用于股票价格预测、气象预报、传感器数据分析等时间序列任务。

```python
class TimeSeriesPredictor(nn.Module):
    """时间序列预测模型"""
    def __init__(self, input_dim, hidden_dim, output_dim, num_layers=2):
        super(TimeSeriesPredictor, self).__init__()
        self.lstm = nn.LSTM(input_dim, hidden_dim, num_layers,
                           batch_first=True)
        self.fc = nn.Linear(hidden_dim, output_dim)
    
    def forward(self, x):
        # x: (batch, seq_len, input_dim)
        lstm_out, (h_n, c_n) = self.lstm(x)
        # 使用最后一个时间步的输出
        last_out = lstm_out[:, -1, :]  # (batch, hidden_dim)
        output = self.fc(last_out)      # (batch, output_dim)
        return output

# 示例：预测未来1个时间步
model = TimeSeriesPredictor(input_dim=1, hidden_dim=64, output_dim=1)
x = torch.randn(32, 50, 1)  # 32个序列，每个长度50
pred = model(x)
print(f"预测结果形状: {pred.shape}")  # (32, 1)
```

## 小结

RNN 及其变体（LSTM、GRU）是处理序列数据的核心工具。RNN 通过隐藏状态捕获时序信息，但存在长期依赖问题；LSTM 通过门控机制有效缓解这一问题；GRU 则在保持性能的同时提高了计算效率。这些模型在语言模型、机器翻译、时间序列预测等领域有广泛应用。
