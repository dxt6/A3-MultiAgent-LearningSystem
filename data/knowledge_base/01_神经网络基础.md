# 神经网络基础

## 1. 神经元模型

人工神经元是神经网络的基本组成单元，其设计灵感来源于生物神经元的信息处理机制。一个典型的人工神经元接收多个输入信号，对每个输入施加权重，求和后通过激活函数产生输出。

数学表达式如下：

\[
z = \sum_{i=1}^{n} w_i x_i + b = \mathbf{w}^T \mathbf{x} + b
\]

\[
a = f(z)
\]

其中：
- \(\mathbf{x} = [x_1, x_2, \ldots, x_n]^T\) 为输入向量
- \(\mathbf{w} = [w_1, w_2, \ldots, w_n]^T\) 为权重向量
- \(b\) 为偏置项（bias）
- \(f(\cdot)\) 为激活函数
- \(a\) 为神经元输出

```python
import numpy as np

class Neuron:
    """单个神经元实现"""
    def __init__(self, input_dim):
        # 初始化权重和偏置
        self.weights = np.random.randn(input_dim) * 0.01
        self.bias = 0.0
    
    def forward(self, x):
        """前向传播"""
        z = np.dot(self.weights, x) + self.bias
        return z

# 示例使用
neuron = Neuron(input_dim=3)
x = np.array([0.5, -0.2, 0.1])
output = neuron.forward(x)
print(f"神经元输出: {output:.4f}")
```

## 2. 激活函数

激活函数引入非线性，使神经网络能够拟合复杂的非线性函数。常见的激活函数包括：

### Sigmoid

\[
\sigma(x) = \frac{1}{1 + e^{-x}}
\]

将输入映射到 \((0, 1)\) 区间，常用于二分类问题的输出层。缺点是容易产生梯度消失。

### Tanh

\[
\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}
\]

将输入映射到 \((-1, 1)\) 区间，输出以零为中心，收敛速度通常比 Sigmoid 快。

### ReLU（Rectified Linear Unit）

\[
\text{ReLU}(x) = \max(0, x)
\]

计算简单，收敛快，是目前最常用的激活函数。缺点是"死神经元"问题（负输入梯度为零）。

### Leaky ReLU

\[
\text{LeakyReLU}(x) = \max(\alpha x, x), \quad \alpha = 0.01
\]

对负输入给予一个小的梯度，缓解"死神经元"问题。

```python
import numpy as np
import matplotlib.pyplot as plt

def sigmoid(x):
    return 1 / (1 + np.exp(-np.clip(x, -500, 500)))

def tanh(x):
    return np.tanh(x)

def relu(x):
    return np.maximum(0, x)

def leaky_relu(x, alpha=0.01):
    return np.where(x > 0, x, alpha * x)

# 绘制激活函数曲线
x = np.linspace(-5, 5, 100)
plt.figure(figsize=(12, 8))
plt.subplot(2, 2, 1); plt.plot(x, sigmoid(x)); plt.title('Sigmoid')
plt.subplot(2, 2, 2); plt.plot(x, tanh(x)); plt.title('Tanh')
plt.subplot(2, 2, 3); plt.plot(x, relu(x)); plt.title('ReLU')
plt.subplot(2, 2, 4); plt.plot(x, leaky_relu(x)); plt.title('Leaky ReLU')
plt.tight_layout()
plt.show()
```

## 3. 前向传播

前向传播是指输入数据从输入层经过隐藏层逐层计算，最终得到输出层结果的过程。以多层感知机（MLP）为例：

假设网络有 \(L\) 层，第 \(l\) 层的计算为：

\[
\mathbf{z}^{(l)} = \mathbf{W}^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}
\]

\[
\mathbf{a}^{(l)} = f^{(l)}(\mathbf{z}^{(l)})
\]

其中 \(\mathbf{a}^{(0)} = \mathbf{x}\) 为输入层，\(\mathbf{a}^{(L)}\) 为最终输出。

```python
import numpy as np

class MLP:
    """简单多层感知机实现"""
    def __init__(self, layer_dims):
        # layer_dims: [输入维度, 隐藏层维度, ..., 输出维度]
        self.parameters = {}
        for i in range(1, len(layer_dims)):
            # He 初始化
            self.parameters[f'W{i}'] = np.random.randn(
                layer_dims[i], layer_dims[i-1]
            ) * np.sqrt(2.0 / layer_dims[i-1])
            self.parameters[f'b{i}'] = np.zeros((layer_dims[i], 1))
    
    def relu(self, Z):
        return np.maximum(0, Z)
    
    def softmax(self, Z):
        exp_Z = np.exp(Z - np.max(Z, axis=0, keepdims=True))
        return exp_Z / np.sum(exp_Z, axis=0, keepdims=True)
    
    def forward(self, X):
        """前向传播"""
        cache = {'A0': X}
        L = len(self.parameters) // 2
        
        # 隐藏层（ReLU激活）
        for l in range(1, L):
            W = self.parameters[f'W{l}']
            b = self.parameters[f'b{l}']
            Z = np.dot(W, cache[f'A{l-1}']) + b
            cache[f'A{l}'] = self.relu(Z)
            cache[f'Z{l}'] = Z
        
        # 输出层（Softmax激活）
        W = self.parameters[f'W{L}']
        b = self.parameters[f'b{L}']
        Z = np.dot(W, cache[f'A{L-1}']) + b
        cache[f'A{L}'] = self.softmax(Z)
        cache[f'Z{L}'] = Z
        
        return cache[f'A{L}'], cache

# 示例使用
mlp = MLP([784, 256, 128, 10])  # 用于MNIST分类
X = np.random.randn(784, 32)     # 32个样本
output, _ = mlp.forward(X)
print(f"输出形状: {output.shape}")  # (10, 32)
```

## 4. 损失函数

损失函数（Loss Function）用于衡量模型预测值与真实值之间的差异，是神经网络训练优化的目标。不同类型的任务需要选择不同的损失函数。

### 均方误差（MSE）

适用于回归问题：

\[
\text{MSE} = \frac{1}{N} \sum_{i=1}^{N} (y_i - \hat{y}_i)^2
\]

```python
def mse_loss(y_true, y_pred):
    return np.mean((y_true - y_pred) ** 2)
```

### 交叉熵损失（Cross-Entropy Loss）

适用于分类问题。对于二分类：

\[
\text{BCE} = -\frac{1}{N} \sum_{i=1}^{N} [y_i \log(\hat{y}_i) + (1 - y_i) \log(1 - \hat{y}_i)]
\]

对于多分类：

\[
\text{CE} = -\frac{1}{N} \sum_{i=1}^{N} \sum_{c=1}^{C} y_{i,c} \log(\hat{y}_{i,c})
\]

```python
def binary_cross_entropy(y_true, y_pred, epsilon=1e-12):
    """二分类交叉熵"""
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))

def categorical_cross_entropy(y_true, y_pred, epsilon=1e-12):
    """多分类交叉熵"""
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.mean(np.sum(y_true * np.log(y_pred), axis=0))

# 示例使用
y_true = np.array([1, 0, 0, 1])  # 二分类真实标签
y_pred = np.array([0.9, 0.1, 0.2, 0.8])  # 预测概率
loss = binary_cross_entropy(y_true, y_pred)
print(f"二分类交叉熵损失: {loss:.4f}")
```

## 小结

神经元是神经网络的基本单元，通过权重和偏置对输入进行线性变换，再经激活函数引入非线性。前向传播将输入逐层传递到输出，损失函数则量化预测误差，为反向传播提供优化目标。理解这些基础概念是深入学习卷积神经网络、循环神经网络等高级架构的前提。
