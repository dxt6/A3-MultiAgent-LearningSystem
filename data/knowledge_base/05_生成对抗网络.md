# 生成对抗网络（GAN）

## 1. GAN 基本原理

生成对抗网络（Generative Adversarial Network, GAN）由 Ian Goodfellow 等人于 2014 年提出，是一种基于博弈论思想的生成模型。GAN 通过两个神经网络的对抗训练，使生成器能够产生逼真的数据样本。

### 核心思想

GAN 包含一个**生成器（Generator）**和一个**判别器（Discriminator）**，二者进行零和博弈：

- **生成器 \(G\)**：试图将随机噪声 \(z\) 转换为逼真的数据样本 \(G(z)\)，以"欺骗"判别器。
- **判别器 \(D\)**：试图区分输入数据是来自真实数据分布 \(x \sim p_{data}\) 还是生成器产生的伪造样本 \(G(z)\)。

### 价值函数（目标函数）

\[
\min_G \max_D V(D, G) = \mathbb{E}_{\mathbf{x} \sim p_{data}(\mathbf{x})}[\log D(\mathbf{x})] + \mathbb{E}_{\mathbf{z} \sim p_{\mathbf{z}}(\mathbf{z})}[\log(1 - D(G(\mathbf{z})))]
\]

其中：
- 判别器 \(D\) 试图最大化上述目标（使 \(D(x)\) 接近 1，\(D(G(z))\) 接近 0）
- 生成器 \(G\) 试图最小化上述目标（使 \(D(G(z))\) 接近 1）

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class Generator(nn.Module):
    """简单生成器（用于生成 28x28 图像，如 MNIST）"""
    def __init__(self, latent_dim=100, img_shape=(1, 28, 28)):
        super(Generator, self).__init__()
        self.img_shape = img_shape
        self.fc = nn.Sequential(
            nn.Linear(latent_dim, 256),
            nn.BatchNorm1d(256),
            nn.ReLU(inplace=True),
            nn.Linear(256, 512),
            nn.BatchNorm1d(512),
            nn.ReLU(inplace=True),
            nn.Linear(512, 1024),
            nn.BatchNorm1d(1024),
            nn.ReLU(inplace=True),
            nn.Linear(1024, int(np.prod(img_shape))),
            nn.Tanh()  # 将输出映射到 [-1, 1]
        )
    
    def forward(self, z):
        img_flat = self.fc(z)
        img = img_flat.view(img_flat.size(0), *self.img_shape)
        return img

class Discriminator(nn.Module):
    """简单判别器"""
    def __init__(self, img_shape=(1, 28, 28)):
        super(Discriminator, self).__init__()
        self.fc = nn.Sequential(
            nn.Linear(int(np.prod(img_shape)), 512),
            nn.LeakyReLU(0.2, inplace=True),
            nn.Linear(512, 256),
            nn.LeakyReLU(0.2, inplace=True),
            nn.Linear(256, 1),
            nn.Sigmoid()  # 输出概率值 [0, 1]
        )
    
    def forward(self, img):
        img_flat = img.view(img.size(0), -1)
        validity = self.fc(img_flat)
        return validity

# 初始化模型
latent_dim = 100
generator = Generator(latent_dim=latent_dim)
discriminator = Discriminator()

# 测试前向传播
batch_size = 32
z = torch.randn(batch_size, latent_dim)
fake_imgs = generator(z)
validity = discriminator(fake_imgs)
print(f"生成图像形状: {fake_imgs.shape}")  # (32, 1, 28, 28)
print(f"判别结果形状: {validity.shape}")    # (32, 1)
```

## 2. 生成器（Generator）

生成器的目标是学习真实数据的分布 \(p_{data}\)，从而能够生成与真实数据难以区分的样本。

### 输入：潜在向量（Latent Vector）

生成器的输入 \(z\) 通常从简单的分布（如均匀分布 \(U(-1, 1)\) 或标准正态分布 \(N(0, I)\)）中采样。潜在空间的结构决定了生成样本的特征变化。

### 网络结构演进

- **原始 GAN**：使用全连接网络，适合生成低分辨率图像。
- **DCGAN**（Deep Convolutional GAN）：使用卷积转置（Transposed Convolution）进行上采样，能够生成更高质量的图像。
- **StyleGAN**：引入风格控制，实现对生成图像细粒度特征的精确控制。

```python
class DCGAN_Generator(nn.Module):
    """DCGAN 生成器（卷积结构）"""
    def __init__(self, latent_dim=100, img_channels=1, feature_maps=64):
        super(DCGAN_Generator, self).__init__()
        self.main = nn.Sequential(
            # 输入: (latent_dim, 1, 1)
            nn.ConvTranspose2d(latent_dim, feature_maps * 8,
                               4, 1, 0, bias=False),
            nn.BatchNorm2d(feature_maps * 8),
            nn.ReLU(True),
            # 输出: (feature_maps*8, 4, 4)
            nn.ConvTranspose2d(feature_maps * 8, feature_maps * 4,
                               4, 2, 1, bias=False),
            nn.BatchNorm2d(feature_maps * 4),
            nn.ReLU(True),
            # 输出: (feature_maps*4, 8, 8)
            nn.ConvTranspose2d(feature_maps * 4, feature_maps * 2,
                               4, 2, 1, bias=False),
            nn.BatchNorm2d(feature_maps * 2),
            nn.ReLU(True),
            # 输出: (feature_maps*2, 16, 16)
            nn.ConvTranspose2d(feature_maps * 2, feature_maps,
                               4, 2, 1, bias=False),
            nn.BatchNorm2d(feature_maps),
            nn.ReLU(True),
            # 输出: (feature_maps, 32, 32)
            nn.ConvTranspose2d(feature_maps, img_channels,
                               4, 2, 1, bias=False),
            nn.Tanh()
            # 输出: (img_channels, 64, 64)
        )
        self.latent_dim = latent_dim
    
    def forward(self, z):
        z = z.view(z.size(0), self.latent_dim, 1, 1)
        return self.main(z)

# 示例使用
gen = DCGAN_Generator(latent_dim=100, img_channels=3, feature_maps=64)
z = torch.randn(8, 100)
fake_images = gen(z)
print(f"DCGAN 生成图像形状: {fake_images.shape}")  # (8, 3, 64, 64)
```

## 3. 判别器（Discriminator）

判别器是一个二分类器，其目标是准确区分真实样本和生成样本。在训练过程中，判别器的性能提升会反过来推动生成器产生更逼真的样本。

### 判别器的训练

判别器的损失函数为二元交叉熵：

\[
\mathcal{L}_D = -\mathbb{E}_{\mathbf{x} \sim p_{data}}[\log D(\mathbf{x})] - \mathbb{E}_{\mathbf{z} \sim p_{\mathbf{z}}}[\log(1 - D(G(\mathbf{z})))]
\]

```python
class DCGAN_Discriminator(nn.Module):
    """DCGAN 判别器"""
    def __init__(self, img_channels=1, feature_maps=64):
        super(DCGAN_Discriminator, self).__init__()
        self.main = nn.Sequential(
            # 输入: (img_channels, 64, 64)
            nn.Conv2d(img_channels, feature_maps, 4, 2, 1, bias=False),
            nn.LeakyReLU(0.2, inplace=True),
            # 输出: (feature_maps, 32, 32)
            nn.Conv2d(feature_maps, feature_maps * 2, 4, 2, 1, bias=False),
            nn.BatchNorm2d(feature_maps * 2),
            nn.LeakyReLU(0.2, inplace=True),
            # 输出: (feature_maps*2, 16, 16)
            nn.Conv2d(feature_maps * 2, feature_maps * 4, 4, 2, 1, bias=False),
            nn.BatchNorm2d(feature_maps * 4),
            nn.LeakyReLU(0.2, inplace=True),
            # 输出: (feature_maps*4, 8, 8)
            nn.Conv2d(feature_maps * 4, feature_maps * 8, 4, 2, 1, bias=False),
            nn.BatchNorm2d(feature_maps * 8),
            nn.LeakyReLU(0.2, inplace=True),
            # 输出: (feature_maps*8, 4, 4)
            nn.Conv2d(feature_maps * 8, 1, 4, 1, 0, bias=False),
            nn.Sigmoid()  # 输出真/假概率
        )
    
    def forward(self, x):
        return self.main(x).view(-1, 1)

# 判别器使用示例
disc = DCGAN_Discriminator(img_channels=3, feature_maps=64)
real_images = torch.randn(8, 3, 64, 64)  # 模拟真实图像
fake_images = torch.randn(8, 3, 64, 64)  # 模拟生成图像

real_validity = disc(real_images)
fake_validity = disc(fake_images)
print(f"真实图像判别结果: {real_validity.shape}")  # (8, 1)
print(f"生成图像判别结果: {fake_validity.shape}")  # (8, 1)
```

### 判别器的关键设计

1. **LeakyReLU 激活函数**：避免"死神经元"问题，在负区间给予小梯度。
2. **批归一化（BatchNorm）**：稳定训练，加速收敛。
3. **Sigmoid 输出层**：输出范围在 \([0, 1]\)，表示样本的"真实"概率。

## 4. GAN 的训练过程

GAN 的训练是一个交替优化的过程，需要在生成器和判别器之间取得平衡。

### 训练步骤

1. **训练判别器**：固定生成器，用真实图像和生成图像训练判别器（最大化判别准确率）。
2. **训练生成器**：固定判别器，更新生成器参数使判别器将生成图像判为"真"（最小化 \(\log(1 - D(G(z)))\)）。

```python
def train_gan(generator, discriminator, dataloader, epochs=50, lr=0.0002):
    """GAN 训练循环（简化版）"""
    import torch.optim as optim
    
    # 优化器
    optimizer_G = optim.Adam(generator.parameters(), lr=lr, betas=(0.5, 0.999))
    optimizer_D = optim.Adam(discriminator.parameters(), lr=lr, betas=(0.5, 0.999))
    
    # 损失函数
    adversarial_loss = nn.BCELoss()
    latent_dim = 100
    
    for epoch in range(epochs):
        for i, (real_imgs, _) in enumerate(dataloader):
            batch_size = real_imgs.size(0)
            
            # 标签
            valid = torch.ones(batch_size, 1)
            fake = torch.zeros(batch_size, 1)
            
            # ====================
            #  训练判别器
            # ====================
            optimizer_D.zero_grad()
            # 真实图像的损失
            real_loss = adversarial_loss(discriminator(real_imgs), valid)
            # 生成图像的损失
            z = torch.randn(batch_size, latent_dim)
            fake_imgs = generator(z)
            fake_loss = adversarial_loss(discriminator(fake_imgs.detach()), fake)
            # 总判别器损失
            d_loss = (real_loss + fake_loss) / 2
            d_loss.backward()
            optimizer_D.step()
            
            # ====================
            #  训练生成器
            # ====================
            optimizer_G.zero_grad()
            z = torch.randn(batch_size, latent_dim)
            fake_imgs = generator(z)
            # 生成器希望判别器将生成图像判为"真"
            g_loss = adversarial_loss(discriminator(fake_imgs), valid)
            g_loss.backward()
            optimizer_G.step()
            
            if i % 100 == 0:
                print(f"[Epoch {epoch}/{epochs}] [Batch {i}] "
                      f"D_loss: {d_loss.item():.4f}, G_loss: {g_loss.item():.4f}")
    
    return generator, discriminator
```

### 训练挑战

1. **模式崩溃（Mode Collapse）**：生成器只产生少数几种样本，缺乏多样性。
2. **训练不稳定**：生成器和判别器的能力需要平衡，否则训练可能发散。
3. **梯度消失**：当判别器过于强大时，生成器接收的梯度趋近于零。

### 改进方案

- **WGAN**（Wasserstein GAN）：使用 Wasserstein 距离替代原始 GAN 的目标函数，引入权重裁剪或梯度惩罚。
- **LSGAN**（Least Squares GAN）：使用最小二乘损失替代交叉熵损失。
- **Conditional GAN**：在生成器和判别器中引入条件信息（如类别标签），实现可控生成。

```python
class ConditionalGAN_Generator(nn.Module):
    """条件 GAN 生成器"""
    def __init__(self, latent_dim=100, num_classes=10, img_shape=(1, 28, 28)):
        super(ConditionalGAN_Generator, self).__init__()
        self.label_emb = nn.Embedding(num_classes, num_classes)
        self.fc = nn.Sequential(
            nn.Linear(latent_dim + num_classes, 256),
            nn.BatchNorm1d(256),
            nn.ReLU(inplace=True),
            nn.Linear(256, 512),
            nn.BatchNorm1d(512),
            nn.ReLU(inplace=True),
            nn.Linear(512, int(np.prod(img_shape))),
            nn.Tanh()
        )
        self.img_shape = img_shape
    
    def forward(self, z, labels):
        # 将噪声和标签拼接
        c = self.label_emb(labels)
        x = torch.cat([z, c], dim=1)
        img_flat = self.fc(x)
        img = img_flat.view(img.size(0), *self.img_shape)
        return img

# 条件生成示例：指定生成数字 "7" 的图像
cgan_gen = ConditionalGAN_Generator(latent_dim=100, num_classes=10)
z = torch.randn(5, 100)
labels = torch.tensor([7] * 5)  # 指定类别
conditioned_images = cgan_gen(z, labels)
```

## 5. GAN 的应用

### 图像生成

GAN 最广为人知的应用是生成逼真图像，如人脸生成（StyleGAN）、艺术图像生成等。

### 图像到图像翻译

**Pix2Pix**：基于条件 GAN 的有监督图像翻译，用于黑白照片上色、草图转照片等任务。

**CycleGAN**：无监督图像翻译，不需要配对的训练数据，用于风格迁移（如马→斑马、照片→莫奈风格）。

### 图像超分辨率

**SRGAN**（Super-Resolution GAN）：通过生成对抗训练，将低分辨率图像重建为高分辨率图像，生成细节更丰富。

### 数据增强

在训练数据不足时，使用 GAN 生成合成数据，提升下游模型的泛化能力。

### 文本到图像生成

**StackGAN**、**AttnGAN**：根据给定的文本描述生成对应的图像。

### 3D 物体生成

**GAN 在 3D 领域**：生成三维模型和点云数据，用于计算机图形学和机器人学。

## 小结

GAN 通过生成器和判别器的对抗训练，能够生成高度逼真的数据样本。生成器负责产生伪造数据，判别器负责区分真假，二者在博弈中共同提升。尽管训练存在挑战（如模式崩溃），但通过 WGAN、条件 GAN 等改进方案，GAN 已在图像生成、图像翻译、超分辨率等多个领域取得令人瞩目的成果，是深度学习生成模型的重要里程碑。
