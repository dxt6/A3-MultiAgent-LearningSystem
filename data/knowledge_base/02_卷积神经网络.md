# 卷积神经网络（CNN）

## 1. 卷积层

卷积层是卷积神经网络的核心组件，通过局部感受野和权值共享机制，高效地提取输入数据的局部特征。与全连接层不同，卷积层中的神经元只与输入数据的局部区域相连。

### 卷积操作

对于输入特征图 \(\mathbf{X}\) 和卷积核 \(\mathbf{K}\)，卷积操作定义为：

\[
(\mathbf{X} * \mathbf{K})_{i,j} = \sum_{m}\sum_{n} \mathbf{X}_{i+m, j+n} \cdot \mathbf{K}_{m,n}
\]

### 关键参数

- **卷积核大小（Kernel Size）**：通常为 \(3\times3\) 或 \(5\times5\)
- **步长（Stride）**：卷积核每次滑动的像素数
- **填充（Padding）**：在输入边缘填充零，保持输出尺寸
- **输出尺寸计算**：
  \[
  \text{输出尺寸} = \left\lfloor \frac{\text{输入尺寸} + 2 \times \text{Padding} - \text{Kernel Size}}{\text{Stride}} \right\rfloor + 1
  \]

```python
import numpy as np
import torch
import torch.nn as nn

# 使用 PyTorch 构建卷积层
conv = nn.Conv2d(
    in_channels=3,       # 输入通道数（如RGB图像）
    out_channels=16,     # 输出通道数（卷积核个数）
    kernel_size=3,       # 卷积核大小
    stride=1,            # 步长
    padding=1            # 填充
)

# 手动实现卷积操作（简化版）
def conv2d_simple(x, kernel, stride=1, padding=0):
    """
    简单2D卷积实现
    x: 输入特征图 (C, H, W)
    kernel: 卷积核 (C, kH, kW)
    """
    if padding > 0:
        x = np.pad(x, ((0,0), (padding,padding), (padding,padding)), 
                   mode='constant')
    
    C, H, W = x.shape
    kH, kW = kernel.shape[1], kernel.shape[2]
    
    out_H = (H - kH) // stride + 1
    out_W = (W - kW) // stride + 1
    output = np.zeros((out_H, out_W))
    
    for i in range(out_H):
        for j in range(out_W):
            region = x[:, i*stride:i*stride+kH, j*stride:j*stride+kW]
            output[i, j] = np.sum(region * kernel)
    
    return output

# 示例：对单通道图像进行边缘检测
image = np.random.randn(1, 28, 28)
edge_kernel = np.array([[1, 0, -1],
                        [1, 0, -1],
                        [1, 0, -1]]).reshape(1, 3, 3)
edge_map = conv2d_simple(image, edge_kernel, padding=1)
print(f"边缘检测结果形状: {edge_map.shape}")
```

## 2. 池化层

池化层（Pooling Layer）用于降低特征图的空间尺寸，减少参数数量和计算量，同时提高模型的平移不变性。常见的池化操作包括最大池化和平均池化。

### 最大池化（Max Pooling）

在局部区域内取最大值，能够保留最显著的特征响应。

\[
y_{i,j} = \max_{(m,n) \in \text{窗口}} x_{i+m, j+n}
\]

### 平均池化（Average Pooling）

在局部区域内取平均值，更平滑地降采样。

```python
import torch.nn as nn
import numpy as np

# PyTorch 中的池化层
max_pool = nn.MaxPool2d(kernel_size=2, stride=2)
avg_pool = nn.AvgPool2d(kernel_size=2, stride=2)

# 全局平均池化（GAP）- 用于替代全连接层
gap = nn.AdaptiveAvgPool2d((1, 1))

# 手动实现最大池化
def max_pool2d_simple(x, pool_size=2, stride=2):
    """
    简单最大池化实现
    x: 输入特征图 (C, H, W)
    """
    C, H, W = x.shape
    out_H = H // pool_size
    out_W = W // pool_size
    output = np.zeros((C, out_H, out_W))
    
    for c in range(C):
        for i in range(out_H):
            for j in range(out_W):
                region = x[c,
                          i*pool_size:(i+1)*pool_size,
                          j*pool_size:(j+1)*pool_size]
                output[c, i, j] = np.max(region)
    
    return output

# 示例
feature_map = np.random.randn(16, 28, 28)
pooled = max_pool2d_simple(feature_map, pool_size=2, stride=2)
print(f"池化前形状: {feature_map.shape}")
print(f"池化后形状: {pooled.shape}")  # (16, 14, 14)
```

## 3. CNN 经典架构

### LeNet-5（1998）

最早的卷积神经网络之一，用于手写数字识别。结构：输入 → 卷积 → 池化 → 卷积 → 池化 → 全连接 → 输出。

### AlexNet（2012）

ImageNet 竞赛冠军，标志着深度学习时代的开启。创新点：
- 使用 ReLU 激活函数
- 引入 Dropout 防止过拟合
- 使用 GPU 进行并行训练

### VGGNet（2014）

统一使用 \(3\times3\) 小卷积核堆叠，证明了增加网络深度可以提升性能。

### ResNet（2015）

引入残差连接（Skip Connection），解决了深层网络梯度消失问题，使网络深度突破 1000 层。

\[
\mathbf{y} = \mathcal{F}(\mathbf{x}) + \mathbf{x}
\]

```python
import torch.nn as nn

class ResidualBlock(nn.Module):
    """ResNet 残差块"""
    def __init__(self, in_channels, out_channels, stride=1):
        super(ResidualBlock, self).__init__()
        self.conv1 = nn.Conv2d(in_channels, out_channels, 
                               kernel_size=3, stride=stride, padding=1)
        self.bn1 = nn.BatchNorm2d(out_channels)
        self.conv2 = nn.Conv2d(out_channels, out_channels,
                               kernel_size=3, stride=1, padding=1)
        self.bn2 = nn.BatchNorm2d(out_channels)
        self.relu = nn.ReLU(inplace=True)
        
        # 当输入和输出维度不匹配时，使用1x1卷积进行维度匹配
        self.shortcut = nn.Sequential()
        if stride != 1 or in_channels != out_channels:
            self.shortcut = nn.Conv2d(in_channels, out_channels,
                                      kernel_size=1, stride=stride)
    
    def forward(self, x):
        identity = self.shortcut(x)
        out = self.relu(self.bn1(self.conv1(x)))
        out = self.bn2(self.conv2(out))
        out += identity  # 残差连接
        out = self.relu(out)
        return out

# 构建一个简单的 ResNet 块
block = ResidualBlock(in_channels=64, out_channels=64)
print(block)
```

### 目标检测

在图像中定位并识别多个目标。代表性算法：YOLO（You Only Look Once）、Faster R-CNN、SSD。

### 图像分割

对图像中每个像素进行分类。代表性算法：FCN（Fully Convolutional Networks）、U-Net、Mask R-CNN。

### 人脸识别

通过 CNN 提取人脸特征，用于身份验证。代表性算法：FaceNet、ArcFace。

### 医学影像分析

CNN 在 CT、MRI 等医学图像的分析中表现出色，用于病灶检测、器官分割等任务。

```python
# U-Net 简化结构（用于图像分割）
class UNetBlock(nn.Module):
    """U-Net 基础卷积块"""
    def __init__(self, in_ch, out_ch):
        super(UNetBlock, self).__init__()
        self.conv = nn.Sequential(
            nn.Conv2d(in_ch, out_ch, 3, padding=1),
            nn.BatchNorm2d(out_ch),
            nn.ReLU(inplace=True),
            nn.Conv2d(out_ch, out_ch, 3, padding=1),
            nn.BatchNorm2d(out_ch),
            nn.ReLU(inplace=True)
        )
    
    def forward(self, x):
        return self.conv(x)

# U-Net 编码器-解码器结构示意
# 编码器：卷积下采样
# 解码器：上采样 + 跳跃连接
# 输出：像素级分割掩码
```

## 小结

卷积神经网络通过卷积层提取局部特征，池化层降低空间分辨率，配合经典架构（如 ResNet）和各类应用场景（分类、检测、分割），已成为计算机视觉领域的核心技术。理解 CNN 的原理和实践，是深入学习和应用深度学习的重要基础。
